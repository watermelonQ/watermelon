面试提醒

  • react 生命周期： construct render componentDidMount shouldComponentUpdate componentDidUpdate componentWillUnMount

  • React Vue 区别：

    React数据的不可变，要通过 setState

    Vue数据可变，实现双向绑定。（组件之间使用的是单向数据）

    React 根组件发生变化时，重新渲染整个组件子树， 避免不必要的子组件的重渲染，需要在所有可能的地方使用 PureComponent，或是实现 shouldComponentUpdate 。 vue 能精确的知道哪块改变了。

    因为开始是写的Vue， React JSX 语法

  • React 定义组件

    函数定义组件: 传 props， memo(component, areEqual) areEqual 没传默认就是浅比较

    类定义: React.Component , React.PureComponent

  • setState(1， 2) 1function or Object 2 callBack

  • react attribute key： When a key changes, React will create a new component instance rather than update the current one. 会调用componentDidMount， eg: componentDidMount 时利用 props.user.id fetch 数据，当 props 变化时，数据不会变化，因为没有重新加载组件。这时候可以监听 componentDidUpdate， 再次 fetch 数据， 或用 props.user.id 当 key，改变时会 create 新的组件

  • Context 不用传在组件树里一级一级传 props 的，用 React.createContext 创建，根组件上 Context.Provider 用的时候 Context.Consumer
