react context

Posted on 2019-01-02 | In react

通过静态方法React.createContext()创建一个Context对象，这个Context对象包含两个组件，<Provider />（生产者）和<Consumer /> （消费者）。

OA 项目的修改 title with context

Index.js 里面 Provider 传一个 setTitle 方法，改变 header 和 document 的title。

有两种情况

  • 已知 title

    写一个高阶组件，一个component + title 返回 一个渲染出title 的组件。

    在高阶组件中 取到consumer，调用provider 中定义的 setTitle，改变title

    （发起流程详情）

  • 动态设置 title

    写一个高阶组件，传一个component，return 一个带有 consumer 包裹的传了setTitle prop的组件。
