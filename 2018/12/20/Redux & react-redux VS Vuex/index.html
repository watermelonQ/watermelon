Redux && react-redux

Posted on 2018-12-20 | In react

###Redux && react-redux

  • Action: 是把数据从应用传到store，通过 store.dispatch(action) 分发，action 传到 store。Action 本质上是 JavaScript 普通对象

    • Action 创建函数 , 返回action

  • Reducer: reducer 就是一个纯函数，接收旧的 state 和 action，返回新的 state。

  • Store: 创建 store

    createStore(reducer, [preloadedState], [enhancer])

    enhancer: function You may optionally specify it to enhance the store with third-party capabilities such as middleware

    • 提供 getState() 方法获取 state；

    • 提供 dispatch(action) 方法更新 state；

    • 通过 subscribe(listener) 注册监听器;

    • 通过 subscribe(listener) 返回的函数注销监听器

  • Redux Middleware

    middleware 最优秀的特性就是可以被链式组合。

    它提供的是位于 action 被发起之后，到达 reducer 之前的扩展点

    可用于进行日志记录、创建崩溃报告、调用异步接口或者路由

    redux-thunk（异步请求）， redux-logger, redux-saga（异步请求）。

  • 容器组件

    • react-redux 中的 connect
