setState(object nextState[, function callback])

合并nextState 和当前 state，（reconcile） ，创建一个即将处理的 state 事务，，相同的周期中多次调用可能会被合并到一起, 可能会批量执行 state 转变（batch update） 。shouldComponentUpdate(), React 创建一个dom树，新的树与老树的节点差异，按需更新。
